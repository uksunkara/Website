# Udaykumar Sunkara - Cybersecurity Analyst Portfolio

This is a personal portfolio website built with React for Udaykumar Sunkara (Cybersecurity Analyst).

## Features

- Modern, professional, and elegant UI
- Sections for About, Projects, Blog, Resume, and Contact
- Responsive, dark-themed design
- Easy to deploy on Vercel, Netlify, or GitHub Pages

## Quick Start

1. Install dependencies:
    ```bash
    npm install
    ```

2. Run locally:
    ```bash
    npm start
    ```

3. Build for production:
    ```bash
    npm run build
    ```

## Customize

- Update content in `src/components/` and `src/data/` as needed.
- Place your resume PDF at `public/resume.pdf`.
- Update social/contact links in `Contact.js`.

## Deployment

- Deploy on [Vercel](https://vercel.com), [Netlify](https://netlify.com), or GitHub Pages.
